### 1. Hardcoded Development URL — BLOCKER
`api_constants.dart` এ local IP আছে:

Production URL দিতে হবে, অথবা environment-based config করতে হবে।

### 2. AuthService এ Duplicate ApiClient তৈরি হচ্ছে
`auth_service.dart` এ:

এটা নতুন ApiClient তৈরি করছে, কিন্তু InitialBinding এ আগেই একটা registered আছে। এটা হওয়া উচিত:

## 🟠 Architecture & Logic Issues

### 6. StorageService — Performance Problem
প্রতিটা call এ নতুন `SharedPreferences.getInstance()` করছে:
```dart
static Future<String> getString(String key) async {
  final preferences = await SharedPreferences.getInstance(); // প্রতিবার!
```
Instance cache করা উচিত:
```dart
static SharedPreferences? _prefs;
static Future<SharedPreferences> get _instance async {
  _prefs ??= await SharedPreferences.getInstance();
  return _prefs!;
}
```

### 7. BottomNabbarBinding — `lazyPut` without `fenix: true`
```dart
Get.lazyPut(() => ChatController());
Get.lazyPut(() => RidesController());
```
যদি controller dispose হয়ে আবার দরকার হয়, recreate হবে না। `fenix: true` দিতে হবে:
```dart
Get.lazyPut(() => ChatController(), fenix: true);
```

### 8. NavigationController — Silent Error Swallowing

```dart
void changeIndex(int index) {

}
```
অন্তত `debugPrint` করা উচিত catch block এ।

### 10. Missing Bindings — অনেক Route এ
